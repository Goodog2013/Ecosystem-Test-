# MB Bank (упрощённый режим)

`MB Bank` в этой версии работает только с профилями и переводами.

Что есть:

- экран входа и регистрации;
- серверные профили;
- перевод между профилями;
- авто-обновление данных между устройствами (короткий poll с сервера);
- встроенный админ-профиль:
  - логин: `Goodog2013`
  - пароль: `Qw111111`

## Запуск

1. Запусти сервер через `start_game.bat`.
2. Открой `http://<IP_сервера>:8000/hub.html`.
3. Выбери проект `МБ Банк`.

Все устройства в одной локальной сети работают с общим состоянием сервера через `/api/mb-bank`.
